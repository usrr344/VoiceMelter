Welcome to VoiceMelter

VoiceMelter is a powerful voice modification tool that allows you to amplify your voice, change the pitch, and configure it for Discord. It's perfect for gamers, streamers, and anyone who wants to enhance their voice in real-time.

Features

  • Real-time voice amplification
  • Pitch control to make your voice higher or lower
  • Easy configuration for Discord
  • Minimize and fullscreen options

Installation

  1. Download the VoiceMelter ZIP file from the link below.
  2. Extract the contents of the ZIP file to a folder on your computer.
  3. Open the folder and double-click on the "VoiceMelter-SETUP" file to start the installation.
  4. Follow the on-screen instructions to complete the installation.

Usage

  1. Launch VoiceMelter from the Start menu or desktop shortcut.
  2. Adjust the amplification factor and pitch using the sliders.
  3. Check the "Listen to yourself" box if you want to hear your modified voice.
  4. Click the "Start" button to begin voice modification.
  5. Configure Discord to use VoiceMelter as the input device.
